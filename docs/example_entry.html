# Der Fließtext

Die Struktur des Fließtextes darf beliebig ausfallen. Die Formatierungsvorgaben basieren dabei auf Markdown, so dass der Text sowohl für Menschen als auch Maschinen lesbar ist. Gute Editoren für Markdown sind etwa Visual Studio Code.

# Dritte Ebene

Die Anzahl der Rauten vor einem Text weist auf die Überschriftenebene hin. Weil der Titel der Webseite selbst die Ebene 1 ist und die Überschrift der Unterseite die Ebene 2, sollten in diesem Fließtext die Überschriften mit Ebene 3 beginnen und somit mit drei Rauten markiert sein. Maximal sind sechs Rauten zulässig.

  • Weitere Details zu Markdown gibt es in Form dieses Cheatsheets.

# Verweise

Verweise sind etwas komplexer. Der Titel des Verweises steht in eckigen Klammern, das Verweisziel in runden Klammern dahinter. Relativen Verweise, etwa auf andere Methoden, müssen mit relativen Pfadangaben erfolgen. Externe Verweisen sollten immer mit http://, https:// oder ohne Angabe des Protokolls :// beginnen.
